FrameLens 사용 방식

FrameLens는 Mac 화면 위에 이동 가능한 투명 캡처 프레임을 표시합니다. 프레임을 이동하거나 크기를 조정하고, 이미지 포맷과 저장 폴더를 선택한 뒤 캡처 버튼 또는 전역 단축키로 프레임 안의 영역을 저장합니다.

  • 1280 x 800, 1440 x 900, 2560 x 1600 및 사용자 지정 크기를 픽셀 기준으로 캡처합니다.
  • PNG, JPEG, HEIC 형식으로 저장할 수 있습니다.
  • 파일명 규칙과 저장 폴더 프리셋을 설정할 수 있습니다.
  • 전역 단축키로 캡처하거나 프레임을 보이기/숨기기 할 수 있습니다.

화면 기록 권한

macOS는 화면 내용을 캡처하는 모든 앱에 화면 기록 권한을 요구합니다. FrameLens는 사용자가 직접 스크린샷을 실행할 때만 이 권한을 사용합니다. 동영상 녹화, 백그라운드 화면 감시, 캡처 이미지 업로드는 하지 않습니다.

권한을 허용하려면 시스템 설정 > 개인정보 보호 및 보안 > 화면 및 시스템 오디오 녹음에서 FrameLens를 허용하세요. 권한 변경 후 앱을 종료하고 다시 열어야 할 수 있습니다.

개인정보 처리방침

FrameLens는 개인 정보를 수집, 저장, 판매, 분석 또는 전송하지 않습니다.

  • 스크린샷은 기본적으로 앱의 샌드박스 컨테이너에 로컬 저장되며, 사용자가 명시적으로 선택한 폴더가 있으면 해당 폴더에 저장됩니다.
  • 설정과 프리셋은 사용자의 Mac에 로컬로 저장됩니다.
  • 분석 도구나 추적 SDK를 사용하지 않습니다.
  • 캡처 이미지를 서버로 업로드하지 않습니다.

자주 묻는 질문

FrameLens가 동영상을 녹화하나요?
아니요. FrameLens는 정지 이미지만 캡처합니다.

정확한 이미지 크기를 지정할 수 있나요?
예. 픽셀 기준 캡처 크기와 사용자 지정 프리셋을 지원합니다.

저장 포맷을 바꿀 수 있나요?
예. PNG, JPEG, HEIC를 지원합니다.

스크린샷은 어디에 저장되나요?
기본적으로 앱의 샌드박스 컨테이너에 로컬 저장됩니다. 설정에서 다른 폴더를 선택하면 사용자가 선택한 해당 폴더에 저장됩니다.
