So funktioniert FrameLens

FrameLens legt einen beweglichen transparenten Aufnahmerahmen über den Mac-Bildschirm. Verschieben oder skalieren Sie den Rahmen, wählen Sie Bildformat und Speicherordner und erfassen Sie den Bereich per Schaltfläche oder globalem Kurzbefehl.

  • Erfassen Sie exakte Größen wie 1280 x 800, 1440 x 900, 2560 x 1600 oder eigene Größen.
  • Speichern Sie als PNG, JPEG oder HEIC.
  • Passen Sie Dateinamenregeln und Ordner-Voreinstellungen an.
  • Nutzen Sie globale Kurzbefehle zum Erfassen oder Ein-/Ausblenden des Rahmens.

Berechtigung für Bildschirmaufnahme

macOS verlangt eine Bildschirmaufnahme-Berechtigung für Apps, die Bildschirminhalte erfassen. FrameLens nutzt diese Berechtigung nur, wenn Sie einen Screenshot auslösen. Die App zeichnet keine Videos auf, überwacht den Bildschirm nicht im Hintergrund und lädt keine Bilder hoch.

Öffnen Sie Systemeinstellungen > Datenschutz & Sicherheit > Bildschirm- und Systemaudioaufnahme und erlauben Sie FrameLens. Danach müssen Sie die App eventuell beenden und neu öffnen.

Datenschutz

FrameLens sammelt, speichert, verkauft, analysiert oder überträgt keine personenbezogenen Daten.

  • Screenshots werden standardmäßig lokal im Sandbox-Container der App gespeichert oder in einem vom Benutzer ausdrücklich gewählten Ordner.
  • Einstellungen und Voreinstellungen werden lokal auf dem Mac gespeichert.
  • Es werden keine Analyse- oder Tracking-SDKs verwendet.
  • Erfasste Bilder werden nicht an Server hochgeladen.

FAQ

Zeichnet FrameLens Videos auf?
Nein. FrameLens erfasst nur Standbilder.

Kann ich die genaue Bildgröße festlegen?
Ja. FrameLens verwendet pixelbasierte Größen und eigene Voreinstellungen.

Kann ich das Ausgabeformat ändern?
Ja. PNG, JPEG und HEIC werden unterstützt.

Wo werden Screenshots gespeichert?
Im lokalen Ordner, der in den Einstellungen ausgewählt wurde.
