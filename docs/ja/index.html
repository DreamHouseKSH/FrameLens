FrameLensの使い方

FrameLensは、Mac画面上に移動可能な透明キャプチャフレームを表示します。フレームを移動またはサイズ変更し、画像形式と保存先フォルダを選択して、ボタンまたはグローバルショートカットでフレーム内の領域を保存します。

  • 1280 x 800、1440 x 900、2560 x 1600、カスタムサイズなどをピクセル単位でキャプチャできます。
  • PNG、JPEG、HEICで保存できます。
  • ファイル名ルールと保存先プリセットをカスタマイズできます。
  • グローバルショートカットでキャプチャ、またはフレームの表示/非表示を切り替えられます。

画面収録権限

macOSでは、画面内容をキャプチャするアプリに画面収録権限が必要です。FrameLensは、ユーザーがスクリーンショットを実行したときだけこの権限を使用します。動画録画、バックグラウンドでの画面監視、キャプチャ画像のアップロードは行いません。

権限を許可するには、システム設定 > プライバシーとセキュリティ > 画面とシステムオーディオ録音を開き、FrameLensを許可してください。変更後にアプリの再起動が必要な場合があります。

プライバシーポリシー

FrameLensは個人データを収集、保存、販売、分析、送信しません。

  • スクリーンショットはデフォルトでアプリのサンドボックスコンテナにローカル保存されます。ユーザーが明示的に別のフォルダを選択した場合は、そのフォルダに保存されます。
  • 設定とプリセットはユーザーのMacにローカル保存されます。
  • 分析ツールやトラッキングSDKは使用しません。
  • キャプチャ画像をサーバーへアップロードしません。

FAQ

FrameLensは動画を録画しますか？
いいえ。FrameLensは静止画像のみをキャプチャします。

正確な画像サイズを指定できますか？
はい。ピクセル単位のキャプチャサイズとカスタムプリセットに対応しています。

出力形式を変更できますか？
はい。PNG、JPEG、HEICに対応しています。

スクリーンショットはどこに保存されますか？
デフォルトではアプリのサンドボックスコンテナにローカル保存されます。設定で別のフォルダを選択した場合は、ユーザーが選択したフォルダに保存されます。
